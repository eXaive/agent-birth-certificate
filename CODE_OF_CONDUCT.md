## 3. Expected Behavior

- Use inclusive and professional language  
- Provide helpful feedback  
- Ask questions respectfully  
- Consider the impact of your words  
- Assume good intent from others  
- Focus on improving the ABC standard  

---

## 4. Unacceptable Behavior

The following are **not permitted**:

- Personal attacks  
- Harassment or discriminatory language  
- Posting private information  
- Threats or intimidation  
- Sabotaging discussions  
- Spamming or derailing technical threads  

---

## 5. Reporting Violations

If you experience or witness unacceptable behavior, you may:

1. Open a confidential issue titled:  
   ```
   Code of Conduct Report
   ```
2. Provide the relevant details.  
3. Maintainers will investigate promptly and fairly.

---

## 6. Enforcement

Maintainers may take appropriate action, including:

- Warnings  
- Removing comments  
- Temporary or permanent bans  
- Reverting harmful contributions  

Enforcement decisions are final.

---

## 7. Scope

This Code of Conduct applies to:

- GitHub issues  
- Pull requests  
- Discussions  
- Documentation  
- Any community interaction related to the ABC standard
